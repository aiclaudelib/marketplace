# aiclaudelib marketplace

Community Claude Code plugins by aiclaudelib.

## Installation

```
/plugin marketplace add aiclaudelib/marketplace
```

## Available Plugins

| Plugin | Description | Category |
|--------|-------------|----------|
| [ccode](https://github.com/aiclaudelib/ccode-plugin) | Create Claude Code skills, agents, hooks, MCP integrations, and plugins with guided workflows, validation, documentation knowledge, and review agents | development |

## Installing a Plugin

After adding the marketplace, install any plugin:

```
/plugin install ccode@aiclaudelib
```
